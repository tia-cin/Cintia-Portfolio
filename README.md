## :checkered_flag: Starting

```bash
# Clone this project
$ git clone https://github.com/tia-cin/Cintia-Portofolio

# Access
$ cd Cintia-Portofolio

# Install dependencies
$ yarn

# Run the project
$ yarn start

# The server will initialize in the <http://localhost:3000>
```
